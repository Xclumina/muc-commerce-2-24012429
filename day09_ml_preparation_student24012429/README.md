## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来做判断的输入信息（如使用月数、投诉次数）。标签是我们要预测的答案（是否流失）。
- 为什么要保留测试集：模拟未见过的真实用户，检验模型有没有真正学到规律（泛化能力）。
- 为什么83%准确率仍可能没有用：因为流失率低，一直预测“不流失”也能有很高准确率，但一个流失用户都抓不到，业务上毫无价值。
